# セキュリティポリシー

## 本パッケージの特性

AnimatorController MA Context は Unity Editor 専用のツールです。ランタイムでは動作せず、ネットワーク通信やユーザー認証の機能も含みません。主な機能はアバター構成情報の読み取りとテキスト出力のみであり、攻撃対象領域は非常に限定的です。

## セキュリティ上の懸念事項

限定的な攻撃対象領域ではありますが、以下のような問題が発見された場合はご報告ください。

- シリアライズ出力に意図せず機密情報が含まれるケース
- 悪意あるアセットを処理した際にエディタがクラッシュまたは不正な動作をするケース
- パッケージの依存関係に起因する脆弱性

## 報告方法

セキュリティに関する問題を発見した場合は、**公開 Issue ではなく**、以下の方法でご報告ください。

1. GitHub の Security Advisory 機能を使用してプライベートに報告してください
2. リポジトリの「Security」タブ → 「Report a vulnerability」から報告できます

## 報告に含めてほしい情報

- 問題の概要
- 再現手順
- 影響範囲の評価
- 可能であれば修正案

## 対応方針

- 報告を受領後、可能な限り速やかに確認・対応します
- 修正が完了するまで、問題の詳細を公開しないようお願いします
- 修正リリース後、報告者の方のご了承を得た上でクレジットを記載します

## サポート対象バージョン

最新リリースバージョンのみセキュリティ修正の対象とします。
